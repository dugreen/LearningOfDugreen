入门RST(reStructuredText)标记文档
================================

章节标题
--------
::
        1.章节头部由下线(也可有上线)和包含标点的标题 组合创建, 其中下线要至少等于标准文本的长度。
        2.可以表示标题的符号有 =、-、`、:、'、"、~、^、_ 、* 、+、 #、<、> 。
        3.对于相同的符号，有上标是一级标题，没有上标是二级标题。
        4.标题最多分六级，可以自由组合使用。
        5.全加上上标或者是全不加上标，使用不同的 6 个符号的标题依次排列，则会依次生成的标题为H1-H6。

        =========
        一级标题
        =========
        二级标题
        =========

        一级标题
        ^^^^^^^^
        二级标题
        ---------
        三级标题
        >>>>>>>>>
        四级标题
        :::::::::
        五级标题
        '''''''''
        六级标题
        """"""""

段落
=====
段落是被空行分割的文字片段，左侧必须对齐（没有空格，或者有相同多的空格）。

缩进的段落被视为引文。

列表
=====
符号列表
>>>>>>>>>
符号列表可以使用 -、 *、+ 来表示。

不同的符号结尾需要加上空行，下级列表需要有空格缩进。
::
        - 符号列表1
        - 符号列表2

         + 二级符号列表1

         - 二级符号列表2

         * 二级符号列表3

        * 符号列表3

        + 符号列表4
